DeemoAce sebenarnya adalah nickname saya dalam dunia game. Nama saya sebenarnya adalah Yudha. Saya sudah suka dunia game sejak SD. Saat itu masih game yang populer adalah game pada PS2. Lalu saat SMP, saya mulai kenal akan dunia perkomputeran. Setelah lulus SMP saya melanjutkan sekolah ke SMK. Di SMK saya mengambil jurusan TAV. Di SMK ini saya mendalami ilmu Teknik Audio Video. Setelah lulus SMK, saya melanjutkan sekolah di Politeknik Kediri. belum sampai semester 3, saya pindah ke Polinema PSDKU kediri.
